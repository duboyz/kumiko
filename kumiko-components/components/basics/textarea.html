Auto-resize Textarea

Textarea that automatically grows as you type. Perfect for descriptions and notes.

Textarea automatically expands as you type
<div class="textarea-group"> <label class="textarea-label">Description</label> <textarea class="textarea textarea-auto" id="auto-textarea"></textarea> </div> <script> function autoResize(textarea) { textarea.style.height = 'auto'; textarea.style.height = textarea.scrollHeight + 'px'; } document.getElementById('auto-textarea').addEventListener('input', function() { autoResize(this); }); </script>

Textarea with Character Counter

Textarea with live character count. Useful for limiting input length.

0 / 200
<div class="textarea-group"> <label class="textarea-label">Description</label> <textarea class="textarea" id="counter-textarea" maxlength="200"></textarea> <div class="textarea-counter" id="char-counter">0 / 200</div> </div> <script> const textarea = document.getElementById('counter-textarea'); const counter = document.getElementById('char-counter'); const maxLength = 200; textarea.addEventListener('input', function() { const length = this.value.length; counter.textContent = length + ' / ' + maxLength; if (length > maxLength * 0.9) { counter.classList.add('warning'); } else { counter.classList.remove('warning'); } }); </script>
