BUTTON

Foundation UI element for user interactions

Icon Button

Small 32x32px square buttons for actions like add, delete, move, etc. Clean and minimal with clear hover states.

<button class="btn-icon">+</button> <button class="btn-icon">×</button> <button class="btn-icon" disabled>+</button>

Primary Button

Main call-to-action button with black background. Used for primary actions like "Save", "Create", "Submit".

<button class="btn-primary">Primary Action</button> <button class="btn-primary" disabled>Disabled</button>

Secondary Button

Secondary action button with white background and border. Used for less important actions like "Cancel", "Edit", "Delete".

<button class="btn-secondary">Secondary Action</button> <button class="btn-secondary" disabled>Disabled</button>

Add Button (Full Width)

Full-width button used for adding new items like "+ Add Item", "+ Add Category". Clean design that fits within containers.

<button class="btn-add">+ Add Item</button> <button class="btn-add" disabled>+ Disabled</button>
